\# APBD Tutorial 2 - Equipment Rental System



\## Project Description

This project is a C# console application that simulates an equipment rental system.

It allows users to rent and return equipment, applies penalties for late returns, tracks unavailable items, and generates a final summary report.



The system includes:

\- different equipment types: Laptop, Projector, Camera

\- different user types: Student and Employee

\- rental management with business rules

\- reporting of active, available, and overdue rentals
